【はじめに】
pngやjpgやgifなどのバイナリを文字列に変換しjQueryで読める
jsを作るツールです。


【使い方】
1. img2js.exeがあるフォルダ内にsourceフォルダを作ります。

2. 1で作ったフォルダの中にtxt化したい画像を置きます。

3. img2js.exeを叩きます。

4. 上手くいくと「img.js」ができます。

5. html上で<img id="#画像名">のようにすると表示されます。
